优势 与
策略梯度

L3 优势 A(advantage)= Q − V　·　∇J = E[∇logπ · A]

优势 A(advantage)告诉策略:这个动作比平均好多少;策略梯度照着 A 推 π——好动作概率抬上去,差动作压下去。这一步跨过去,就摸到 PPO 的门了。

Figure · advantage 推动 policy
梯度方向 = ∇logπ(a│s) · A(s,a)　→　A>0 抬 / A<0 压
①

优势 A = Q − V

Q 是「这个动作的好」,V 是「这个状态所有动作的平均好」。相减 = 这个动作比平均超出多少。

📄 GAE 论文 p.2(A=Q−V 定义)📖 S&B 接 L2 的桥 V=Σπ·Q
"The advantage function, by its definition A^π(s,a) = Q^π(s,a) − V^π(s), measures whether or not the action is better or worse than the policy's default behavior."
— Schulman et al., GAE (ICLR 2016), p.2
Aπ(s,a) = Qπ(s,a) − Vπ(s)
接 L2 例子:Q(a₁)=8, Q(a₂)=3, V=6.5 → A(a₁)=+1.5,A(a₂)=−3.5

A 的期望为 0:它本就是「相对平均」的量

这不是新性质,是定义的直接推论。V 就是 Q 在 π 下的平均(L2 桥 V=Σ_a π(a│s)Q(s,a));A=Q−V=「比平均高多少」——任何「偏离平均」的量,平均自然为 0(像考试分减班级平均分)。所以从「定义 A」到「期望为 0」不是跳步,是同一句话。

精确写:E_{a~π(·│s)}[Aπ(s,a)] = E[Q] − V = V − V = 0
↑ 是「固定状态 s、动作 a 按 π 采样」的期望,对每个 s 都成立;不是对所有 (s,a) 均匀平均。「零均值」是它的口语说法。

验证(L2 的数)0.7×(+1.5) + 0.3×(−3.5) = 1.05 − 1.05 = 0 ✓
为什么要看它A = 「Q 减基线 V」。期望为 0 ⇒ 减基线不引入偏差;居中于 0 ⇒ 数值小、方差小、训练稳。
②

实践中怎么估 A:TD 误差 δ

承上:① 给的 A=Q−V 只是「定义」,不等于「能算」——要得出具体的 A 得有 Q、V 的数值,而真实 Q 拿不到。于是退一步:用 V 来估 A。下面三步拆。

📖 S&B §11.1 p.258(actor-critic δ)📄 GAE 式(10)
从「Q 拿不到」到「用 δ 估 A」· 三步
① 为什么真实 Q 拿不到Q(s,a)=E[G│s,a] 是对「所有可能的未来」取的期望——未来无穷多、环境/策略带随机;你既没有环境模型(model-free)、也跑不完每条未来,只看得到有限采样 → 只能估。(LLM 里更直接:一个 prompt 的续写有天文数字种,你只采几条。)
② 那 V 呢?V 也估,但便宜得多V 只依赖状态 s(不依赖动作),是个更小的函数,可由一个 critic 网络从数据里学出近似 V̂。
③ 关键技巧:有了 V,不碰 Q 也能估 A走一步、看到真实奖励 r,r+γV(s′) 比 V(s) 更准;两者之差就是 TD 误差 δ,而 δ 恰好是一步 advantage 的估计。为什么相等?↓ 下面拆。
TD 是什么 · Temporal Difference 时序差分

Temporal(相邻时刻)+ Difference(两个估计之差):用「走一步后的更好估计」回头纠正「走之前的估计」。δ 就是这两个估计相减。

旧估计V(sₜ)——走之前对价值的猜测
新估计 = TD targetrₜ + γV(sₜ₊₁)——看到真实 r 和下一格后的更好猜测

δ = 新估计 − 旧估计 = 旧猜测错了多少。核心是 bootstrapping:不等结局、走一步就用「自己下一步的估计」更新;对比蒙特卡洛要等整局结束用真实回报 G(无偏但慢、方差大)。

📖 TD 学习是 Sutton 1988 的奠基贡献;δ 公式见 S&B §11.1 p.258

δt = rt + γV(st+1) − V(st)
V 准确时,E[δ] = Q − V = A(GAE 式 10)
"If the TD error is positive, it suggests that the tendency to select Aₜ should be strengthened for the future, whereas if the TD error is negative, it suggests the tendency should be weakened."
— Sutton & Barto, §11.1 actor–critic, p.258

δ 的直觉

δ>0:这一步比预期好(actor 把这个动作概率抬高);δ<0:比预期差(压低)。这正是 advantage 的符号。

critic 学 V,actor 用 δ 调 π —— 这就是 actor-critic,也是 PPO 里 Critic 的活。

③

策略梯度:照着 A 推 π

把 A 当权重,乘上 ∇logπ,就是更新策略的梯度。一句话:好动作概率抬高,差动作压低。

📄 GAE 式(9) p.3📄 Sutton-PG-1999(策略梯度定理来源)
ĝ = E[ Â(s,a) · ∇θ log πθ(a│s) ]
A>0 → 把 logπ(a│s) 往上推(多做);A<0 → 往下推(少做)
"a step in the policy gradient direction should increase the probability of better-than-average actions and decrease the probability of worse-than-average actions. ... The choice Ψₜ = A^π(sₜ,aₜ) yields almost the lowest possible variance."
— Schulman et al., GAE (ICLR 2016), p.2
为什么用 A 不用 raw return / Q? 用 Q,高价值状态里所有动作权重都大 → 噪声大;减掉基线 V 变成 A(零均值)→ 只奖励「超出平均」的动作 → 方差小、训练稳。这就是 baseline 的全部意义,也是 actor-critic / PPO / GRPO 的共同地基。
④

落到代码:cleanrl 里就这几行

上面的 δ、GAE、A·∇logπ,在真实 PPO 实现里是连续几行。先认 advantage 那段。

💻 cleanrl/ppo.py:229–231(GAE)💻 :251–252, 265–267(A→loss)
# --- GAE:把 δ 累成 advantage(ppo.py:229–231)---
229 delta = rewards[t] + gamma * nextvalues * nextnonterminal - values[t] # ← δ = r+γV(s')−V(s)
230 advantages[t] = lastgaelam = delta + gamma * gae_lambda * nextnonterminal * lastgaelam # ← GAE(λ) 递推
231 returns = advantages + values # ← target for critic

# --- advantage 当权重,推 logπ(ppo.py:251–267)---
251 logratio = newlogprob - b_logprobs[mb_inds]
252 ratio = logratio.exp() # ← π_new/π_old(PPO 的 ratio,下一课 L4)
265 pg_loss1 = -mb_advantages * ratio # ← −A·(π_new/π_old),核心就是 A·∇logπ
266 pg_loss2 = -mb_advantages * clamp(ratio, 1−ε, 1+ε) # ← clip(L4)
267 pg_loss = max(pg_loss1, pg_loss2).mean()
L3 你只要认住 229–231(把 δ 累成 A) 和 265(−A·ratio,本质是 A·∇logπ)。ratio / clip(252/266)是 PPO 的加料,下一课 L4 专讲。
⑤

GAE(λ):偏差 / 方差的旋钮

把多步 δ 按 (γλ) 衰减加起来,用一个 λ 在「低方差但有偏」和「无偏但高方差」之间调。

📄 GAE 式(11)–(16),λ 定义💻 cleanrl/ppo.py:230(gae_lambda)
ÂGAEt = Σl≥0 (γλ)l δt+l
λ=0 → Â=δ(一步,低方差/有偏)　|　λ=1 → Â=Σγ^l r − V(蒙特卡洛,无偏/高方差)
cleanrl 默认 gae_lambda=0.95(ppo.py:51)——靠近 1 偏向低偏差,但留一点方差控制。λ 就是 advantage 估计的"看多远"旋钮。

Q4策略梯度梯度 ∇J = E[∇logπ · A] 里,A>0 时对该动作做什么?
A>0 → 把 logπ(a│s) 往上推 → 该动作更可能被选。GAE p.2 原话:increase prob of better-than-average actions。
